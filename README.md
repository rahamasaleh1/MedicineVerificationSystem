# Medicine Verification System

## Overview
This project simulates a simple medicine verification system. It allows a user to input a medicine ID, view medicine details, and check whether it has been flagged as counterfeit.

## Problem
Counterfeit medicines present serious risks. There is a need for systems that help pharmacists and regulators verify medicine authenticity quickly and reliably.

## Solution
This project uses a CSV dataset to simulate a verification system. Users can enter a medicine ID and the system will return relevant information and flag counterfeit products.

## Features
- Search medicine by ID
- Display medicine details
- Flag counterfeit medicines
- Log scan history

## Tools Used
- Python
- CSV

## Files
- app.py
- medicines.csv

## How to Run
Run the following in terminal:

python app.py

Then enter a medicine ID such as:
MED003
